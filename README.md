## 概要

「ブラウザ版 タイプウェル国語 R」は、GANGAS 様制作「タイプウェル国語 R」風のタイピング練習アプリです。

本家と比べるとかなり機能は少ないです（タイムとレベルを計測できる程度で、詳細結果やランキングといった機能はありません）。

### 動作確認済み環境

- Windows 10
  - Google Chrome
  - Vivaldi
  - Microsoft Edge
- Ubuntu 18.04(LTS)
  - Google Chrome
- Android 9（Bluetooth キーボード接続時）
  - Google Chrome
